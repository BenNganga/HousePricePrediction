# Customer Churn Prediction

A Machine Learning project that predicts whether a customer is likely to leave a company based on customer information.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## Project Structure

- data/
- models/
- notebooks/
- src/
- app.py
- main.py

## Run

```bash
pip install -r requirements.txt
python main.py
```
